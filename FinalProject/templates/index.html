Book Tracking Website

At the beginning of this project, my plans looked like this. I used the google books API to allow users to search for books, and I created a secure login system. Users can create accounts and use them to store books they want to read. I also added a page with my personal recommendations for various genres, as well as inserted a Goodreads widget with my reading updates. To store my recommendations, I created a SQL database to store genres and books. For some books, there are reviews that I have written that show up when you search for them.
